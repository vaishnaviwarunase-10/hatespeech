# Hate Speech Detection

This project detects hate speech in text using Machine Learning.

## Features
- Text preprocessing
- Model training
- Prediction system

## Files
- app.py → main app
- model_training.py → training model
- model.pkl → trained model
- dataset.csv → dataset

## Tech Used
- Python
- Scikit-learn
- Pandas

## How to Run
1. Install requirements
2. Run app.py
